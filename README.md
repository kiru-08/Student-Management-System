# 🎓 Student Management System

A responsive Student Management System developed using Flask, Python, MySQL, Bootstrap, Chart.js, and OpenPyXL.

## 🚀 Features

- 🔐 Admin Login
- 👨‍🎓 Add Student
- ✏️ Edit Student
- 🗑 Delete Student
- 🔍 Search Students
- 📊 Dashboard Analytics
- 📈 Reports with Charts
- 📥 Export to Excel
- 🗄 MySQL Database Integration
- 📱 Responsive UI

## 🛠 Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Chart.js
- OpenPyXL
